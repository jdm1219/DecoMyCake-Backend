# 내 케이크를 꾸며줘 🍰

다른 사람의 케이크를 꾸며보세요.

## 🔥 Install

### 📦 Package

```bash
$ npm install
```

### 💾 Prisma

```bash
$ npx prisma
$ prisma init
$ prisma generate
```

## 🚀 Run

```bash
$ npm run start # development
$ npm run start:dev # watch mode
$ npm run start:prod # production mode
```

## 📽 Build

```bash
$ npm run build
```

## 🛠 Dependency

[Nest](https://nestjs.com)
